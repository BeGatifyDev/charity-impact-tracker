== Description ==

**Charity Impact Tracker** is a professional WordPress plugin that enables charities and non-profits to record, manage, and showcase their impact in an organized, visually appealing manner. 

Key features include:

- Custom Post Types for recording charity impacts.
- Easy-to-use shortcodes for frontend display.
- Salesforce and external API integrations for automated data updates.
- Fully responsive and accessible frontend layouts.
- Modular architecture for easy extension.

Developed with love by **Oluwafemi Oluwatobi Best** to empower NGOs, social enterprises, and humanitarian brands with world-class digital tools.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Charity Impact Tracker** in your dashboard menu.
4. Configure settings and add impacts as needed.

== Frequently Asked Questions ==
